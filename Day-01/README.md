# Day 01 - C Programming Fundamentals

## Overview

This directory contains the programs completed on **Day 01** of my C Programming Practice Journey.

The objective of Day 01 was to build a strong foundation in:

* C Program Structure
* Variables and Data Types
* Input and Output Functions
* Arithmetic Operations
* Type Casting
* Character Handling
* Mathematical Calculations
* Basic Problem Solving

All programs are written in a simple and beginner-friendly manner with proper comments, sample inputs, and sample outputs for better understanding and future reference.

---

## Programs Completed

| Program No. | Program Name                       | Description                                                                                      |
| ----------- | ---------------------------------- | ------------------------------------------------------------------------------------------------ |
| 01          | Hello World                        | Displays a simple message on the console.                                                        |
| 02          | Greeting Program                   | Accepts the user's name and displays a welcome message.                                          |
| 03          | Arithmetic Operations              | Performs addition, subtraction, multiplication, division, and modulus operations on two numbers. |
| 04          | Swapping Two Numbers               | Swaps two numbers using a temporary variable.                                                    |
| 05          | Area and Circumference of a Circle | Calculates the area and circumference of a circle using the radius provided by the user.         |
| 06          | Area and Perimeter of a Rectangle  | Calculates the area and perimeter of a rectangle using length and breadth.                       |
| 07          | Simple Interest Calculator         | Computes simple interest based on principal amount, rate of interest, and time period.           |
| 08          | Average of Three Numbers           | Calculates the average of three integer values.                                                  |
| 09          | Celsius to Fahrenheit Conversion   | Converts temperature from Celsius to Fahrenheit.                                                 |
| 10          | ASCII Value of a Character         | Displays the ASCII value corresponding to a given character.                                     |

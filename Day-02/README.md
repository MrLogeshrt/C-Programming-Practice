# Day 02 - Decision Making Statements

## Overview

This directory contains the programs completed on **Day 02** of my C Programming Practice Journey.

The objective of Day 02 was to strengthen problem-solving and logical thinking skills using:

* Decision Making Statements
* Nested Conditions
* Relational Operators
* Logical Operators
* Input Validation
* Data Classification
* Business Rule Validation
* Real-World Problem Solving

All programs are written in a structured and beginner-friendly manner with proper comments, sample inputs, and sample outputs for better understanding and future reference.

---

## Programs Completed

| Program No. | Program Name | Description |
|------------|-------------|-------------|
| 11 | Number Classification | Determines whether a given integer is positive, negative, or zero and identifies it as odd or even. |
| 12 | Largest of Three Numbers | Determines the largest among three integers while properly handling equality cases. |
| 13 | Triangle Type Identifier | Validates triangle side lengths and identifies whether the triangle is Equilateral, Isosceles, or Scalene. |
| 14 | Leap Year Validator | Determines whether a given year is a leap year based on Gregorian calendar rules. |
| 15 | Character Classifier | Classifies a character as an uppercase letter, lowercase letter, digit, or special character. |
| 16 | Electricity Bill Calculator | Calculates electricity bill amount using slab-based unit consumption rates. |
| 17 | Bank Loan Eligibility Check | Determines loan eligibility based on age, employment status, income, and existing loan amount. |
| 18 | Income Tax Calculator | Calculates income tax using progressive annual income tax slabs. |
| 19 | Student Grade Analyzer | Determines grade and performance category based on the student's score. |
| 20 | ATM Withdrawal Validator | Validates ATM withdrawal requests based on banking constraints and account balance. |

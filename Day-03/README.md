# Day 03 - Switch Case Statements

## Overview

This directory contains the programs completed on **Day 03** of my C Programming Practice Journey.

The objective of Day 03 was to strengthen menu-driven programming skills and learn how to use:

* Switch Case Statements
* Multi-Way Decision Making
* Menu-Driven Applications
* Mathematical Computations
* Information Mapping
* Business Logic Implementation
* Input Validation
* Real-World Problem Solving

All programs are written in a structured and beginner-friendly manner with proper comments, sample inputs, and sample outputs for better understanding and future reference.

## Programs Completed

| Program No. | Program Name | Description |
|------------|-------------|-------------|
| 21 | Scientific Calculator | Performs arithmetic and scientific calculations based on user-selected operations. |
| 22 | Smart Unit Converter | Converts values between different measurement units using a menu-driven approach. |
| 23 | Month Information System | Displays the month name, number of days, and quarter based on the month selected. |
| 24 | ATM Menu Simulator | Simulates ATM operations such as balance inquiry, deposit, and withdrawal using menu options. |
| 25 | Employee Department Information | Displays employee department details based on department selection. |
| 26 | Electricity Appliance Menu | Displays information and power ratings of various household electrical appliances. |
| 27 | Area Calculator | Calculates the area of different geometric shapes based on user selection. |
| 28 | Student Grade Information System | Displays grade details, performance level, and academic status based on grade selection. |
| 29 | Travel Ticket Booking System | Displays travel fare and journey information for different transportation modes. |
| 30 | Product Billing System | Displays product details and calculates the total bill based on quantity purchased. |

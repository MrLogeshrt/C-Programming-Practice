# Day 04 - Loops

## Overview

This directory contains the programs completed on **Day 04** of my C Programming Practice Journey.

The objective of Day 04 was to develop a strong understanding of iterative programming using loops and to apply looping constructs in solving real-world business scenarios. Instead of focusing on traditional textbook problems, the programs were designed to simulate practical applications that strengthen logical thinking, input validation, statistical analysis, and report generation.

The concepts covered include:

* For Loop
* While Loop
* Do-While Loop
* Nested Loops
* Loop Control Statements (`break` and `continue`)
* Counter-Controlled Iteration
* Sentinel-Controlled Iteration
* Running Totals and Accumulators
* Maximum and Minimum Value Tracking
* Statistical Calculations
* Business Logic Implementation
* Input Validation

All programs are written with structured documentation, meaningful variable names, proper comments, and multiple sample test cases to maintain code quality and readability.

## Programs Completed

| Program No. | Program Name | Description |
|-------------|--------------|-------------|
| 31 | ATM PIN Verification | Simulates ATM PIN authentication with limited login attempts and account lock functionality. |
| 32 | Student Attendance Tracker | Calculates attendance percentage and displays attendance status based on daily attendance records. |
| 33 | Factory Production Monitor | Records hourly production units and generates production statistics without storing data in arrays. |
| 34 | Bank Deposit Analysis System | Analyzes daily customer deposits and generates statistical reports including average, highest, and lowest deposits. |
| 35 | Parking Fee Calculator | Calculates parking charges for multiple vehicles based on progressive hourly pricing rules. |
| 36 | Electricity Bill Management System | Calculates electricity bills for multiple consumers and generates a revenue summary using slab-based billing. |
| 37 | Student Result Analysis System | Processes student marks to calculate pass percentage, average marks, and overall class performance. |
| 38 | EMI Collection Management System | Tracks EMI payments, calculates collection percentage, and generates payment reports. |
| 39 | Hospital Patient Waiting Time Analyzer | Analyzes patient waiting times and generates statistical information for hospital queue management. |
| 40 | Library Fine Collection Report | Calculates fines for returned library books and generates fine collection statistics. |

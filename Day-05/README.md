# Day 05 - Number Analysis Using Loops

## Overview

This directory contains the programs completed on **Day 05** of my C Programming Practice Journey.

The objective of Day 05 was to strengthen my understanding of:

* Loop-Based Number Analysis
* Digit Extraction Techniques
* Number Property Verification
* Nested Loops
* Mathematical Computations
* Input Validation
* Real-World Number Analysis Problems

All programs are written in a structured and beginner-friendly manner with proper documentation, meaningful comments, sample inputs, and sample outputs for better understanding and future reference.

---

## Programs Completed

| Program No. | Program Name | Description |
|------------|-------------------------------------|-------------------------------------------------------------------------------------------------------------|
| 41 | Employee ID Validator | Analyzes an employee ID by calculating the sum of digits, number of digits, first digit, last digit, and determines whether it is even or odd. |
| 42 | Password Strength Analyzer | Evaluates a numeric password based on digit analysis and classifies its overall strength. |
| 43 | Vehicle Number Analyzer | Analyzes a 4-digit vehicle number by calculating the sum of digits, reverse, palindrome status, and number type. |
| 44 | Account Number Analyzer | Analyzes an account number by determining the digit count, sum, product, highest digit, and lowest digit. |
| 45 | Invoice Number Analyzer | Analyzes an invoice number by counting even and odd digits, identifying highest and lowest digits, and displaying digit frequencies. |
| 46 | Lucky Customer Number Checker | Determines whether a customer number is lucky by comparing the sum of digits in its first and second halves. |
| 47 | Product Number Inspector | Analyzes a product serial number by identifying digit count, highest digit, lowest digit, and displaying digits in sorted order. |
| 48 | Secure Number Analyzer | Determines whether a number is Palindrome, Armstrong, Perfect, and Strong, and classifies its overall security level. |
| 49 | Package Number Analyzer | Calculates the sum, average, reverse, and counts prime and composite digits in a package number. |
| 50 | Banking Security Analyzer | Performs a complete analysis of a banking security number including palindrome, Armstrong, digit statistics, and mathematical properties. |

# Day 06 - Arrays in C Programming

## Overview

This directory contains the programs completed on **Day 06** of my C Programming Practice Journey.

The objective of Day 06 was to understand and apply the concept of **Arrays** for storing and processing multiple data elements efficiently. The programs focus on solving real-world analytical problems using arrays, loops, conditional statements, and statistical calculations.

All programs are written in a simple and beginner-friendly manner with proper comments, sample inputs, and sample outputs for better understanding and future reference.

---

## Programs Completed

| Program No. | Program Name | Description |
| ------------ | ------------------------------------------ | -------------------------------------------------------------------------------------------------------------- |
| 51 | Employee Salary Analytics System | Analyzes employee salary records to determine total salary expenditure, highest salary, lowest salary, average salary, and employees earning above or below the average salary. |
| 52 | Student Performance Dashboard | Evaluates students' academic performance by calculating highest mark, lowest mark, class average, grade distribution, and pass/fail statistics. |
| 53 | Smart Stock Manager | Manages product inventory by analyzing stock levels, identifying out-of-stock, low-stock, and well-stocked products. |
| 54 | ATM Cash Denomination Tracker | Calculates total ATM cash, denomination-wise contribution, highest and lowest denomination count, and total number of notes available. |
| 55 | Patient Temperature Monitor | Analyzes patient body temperatures to determine average temperature, highest, lowest, and classify patients as normal, fever, or critical. |
| 56 | Vehicle Speed Analysis Report | Monitors vehicle speeds by calculating average speed, highest speed, lowest speed, and identifying overspeed vehicles. |
| 57 | House Electricity Consumption Report | Evaluates electricity usage by calculating average consumption, highest and lowest consumption, and identifying low and high consumption houses. |
| 58 | Candidate Evaluation Report | Analyzes candidate scores to determine highest score, lowest score, average score, top performer, qualified candidates, and above-average performers. |
| 59 | Data Packet Validator | Validates packet transmission by checking packet integrity, calculating checksum, identifying corrupted packets, and determining packet validity. |
| 60 | Sensor Stability Analyzer | Analyzes sensor readings by calculating average reading, highest and lowest values, maximum spike, and determining sensor stability status. |
